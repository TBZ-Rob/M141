## 🔁 Wiederkehrende Prompts

### 📐 Darstellung aktualisieren
```
Analysiere das ganze Repo, aktualisiere jedes Diagramm oder Darstellung
auf den neusten Stand und füge bei neuen Seiten hinzu.
```

### ✏️ Rechtschreibung prüfen
```
Prüfe die Rechtschreibung und Grammatik aller bestehenden Seiten.
Verändere dabei den Inhalt und die Aussage des Textes nicht.
```

### 🎨 Design Vorschläge
```
Bringe mir ein paar Vorschläge für ein ansprechendes Design dieser Seite.
Keine grossen Textblöcke, klare Struktur, mit Navigation zu anderen Seiten.
```

### ✅ Design übernehmen
```
Überarbeite das Design genau so wie in Vorschlag [X] beschrieben.
Verändere dabei den Inhalt und die Aussage des Textes nicht.
```
